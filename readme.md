# R.E.S.Q (Rapid Emergency Surveillance & Quenching)


# Problem Statement

Fire response times in Cagayan De Oro City frequently exceed the 7-minute target due to late incident reporting, reliance on manual phone hotlines, and logistical challenges.


# Target Users
    - Residents of Cagayan De Oro City
    - Fire Responders (BFP)
    - Local Government Units (LGUS)
    - System Administrators

# Tech Stack
    - ReactJS (Frontend)

# Documentation

## Getting Started

To run the project locally, follow these steps:

1. **Clone Project:**
   ```bash
   git clone https://github.com/NBlancs/R-E-S-Q-app-dev.git
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
